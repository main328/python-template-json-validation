# Python Template JSON Validation
## 설치 방법

1. Python 3.12 이상 권장.

2. 가상환경 생성:
```bash
python -m venv .venv
```
3. 가상환경 활성화:
```bash
# Windows
.venv\Scripts\activate
# Linux/Mac
source .venv/bin/activate
```

4. 필수 의존성 설치:
```bash
pip install -r requirements.txt
```

5. 가상환경 비활성화.
```bash
deactivate
```
